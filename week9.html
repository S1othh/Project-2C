In Class Activity

This activity is a combination of the use of physical computing, p5.js, Arduino, and HTML. The Arduino reads the turning of a knob (potentiometer) on pin A0 and sends that number over USB every tenth of a second. Also, it can control the lightness of the screen. By adding JavaScript code in p5.js, I can add more aesthetic elements on the computer.

By doing more experiments on using physical computing, Arduino, p5.js, and HTML, I can explore the variety of interactive media.
